# 📊 Análise de Desempenho Escolar com Python

Projeto de análise de dados educacionais desenvolvido com Python, com foco em diagnóstico pedagógico e apoio à tomada de decisão baseada em dados.

---

## 🎯 Objetivo

Analisar o desempenho dos alunos a partir de dados em Excel, identificando padrões de aprendizagem, níveis de proficiência e pontos críticos para intervenção pedagógica.

---

## 🛠️ Tecnologias utilizadas

* Python
* pandas
* matplotlib

---

## 📈 Funcionalidades

* Leitura e tratamento de dados em Excel
* Cálculo de média geral da turma
* Análise de desempenho por disciplina
* Classificação dos alunos por nível (modelo inspirado no SARESP)
* Identificação de alunos abaixo da média
* Geração automática de gráficos
