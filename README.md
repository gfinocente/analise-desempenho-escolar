# 📊 Análise de Dados Educacionais com Python

## 🎯 Objetivo
Este projeto tem como objetivo analisar o desempenho de alunos do 9º ano do Ensino Fundamental a partir de dados do SARESP 2025, com foco em apoiar a tomada de decisão pedagógica baseada em dados.

A proposta é transformar dados brutos em informações estratégicas para professores e gestores, permitindo intervenções mais assertivas no processo de ensino-aprendizagem.

---

## 📚 Contexto
A análise foi realizada com dados reais de aproximadamente **35 alunos**, distribuídos em **1 turma**, refletindo um cenário escolar autêntico.

O projeto foi desenvolvido a partir da necessidade prática de:
- Identificar alunos em risco de aprendizagem
- Mapear dificuldades por disciplina
- Apoiar o planejamento pedagógico com base em evidências

---

## ⚙️ Tecnologias Utilizadas
- Python
- Pandas
- Matplotlib
- Excel (base de dados)

---

## 🔍 Etapas da Análise
- Leitura e tratamento de dados em Excel  
- Cálculo de médias gerais e por disciplina  
- Classificação dos alunos por níveis de desempenho (modelo inspirado no SARESP)  
- Identificação de alunos em risco  
- Geração de gráficos automatizados  

---

## 📈 Principais Insights
- Visualização clara das disciplinas com maior dificuldade  
- Distribuição dos alunos por nível de aprendizagem  
- Identificação rápida de grupos prioritários para intervenção pedagógica  

Essa abordagem permite reduzir o tempo de análise manual e direcionar ações pedagógicas com maior precisão.
